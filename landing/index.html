The squad

Three roles collapse. Twelve roles thrash on coordination. Six is the sweet spot.

Strategist

Reads the brief, scouts existing submissions, picks the angle most likely to win.

PM

Scopes the MVP, enforces the time budget, kills scope creep.

Dev

Writes the code, ships to Vercel. Delegates to Hermes's bundled claude-code skill.

Designer

Brand palette, hero image, project page hero.

Storyteller

Project page HTML, README, demo script.

Submitter

Creates the draft, fills the form, awaits human approval. Never auto-publishes.

How it works

The orchestrator is a markdown skill, not orchestration code. The plugin is six tools wired to Hermes via a single register(ctx) call.

The plugin

plugin/hackclaw/
├── plugin.yaml
├── __init__.py        # register(ctx) entry point
├── tools/
│   ├── hackclaw_select_platform.py
│   ├── hackclaw_get_brief.py
│   ├── hackclaw_list_submissions.py
│   ├── hackclaw_create_draft.py
│   ├── hackclaw_update_project.py
│   └── hackclaw_submit.py
└── platforms/
    ├── base.py        # HackathonPlatform protocol
    ├── taikai.py      # MCP + GraphQL routes
    └── browser.py     # v0.3 stub

Install

Prerequisite: a working Hermes Agent install. 

git clone https://github.com/hypermari0/hackclaw.git
cd hackclaw
./install.sh
pip install -e .

# pick one TAIKAI route
export HACKCLAW_TAIKAI_VIA=mcp        # default, native via MCP
# or
export HACKCLAW_TAIKAI_VIA=graphql
export TAIKAI_TOKEN=...

hermes

Then talk to it:

> hackathon run https://taikai.network/<org>/hackathons/<slug>

Or just say it natural. "win Hacklayer 4_1 for me." Same skill.
